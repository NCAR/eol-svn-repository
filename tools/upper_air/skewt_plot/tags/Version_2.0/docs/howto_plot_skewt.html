How To: Create SkewT Plots from ESC Soundings

Purpose

The purpose of this document is to describe the process of using the plot_skewt.pl script for generating SkewT plots from ESC formatted sounding data.

Repository

The skewT plot script is under revision control using subversion. The repository can be found at:

http://svn.atd.ucar.edu/svn/dmg/tools/upper_air/plot_script

Executing the Script

The script can be run directory from the directory:

/net/work/bin/plotters/plot_skewt.pl

It takes the following sets of parameters. Parameters in italics are optional.

  • -o dir: The output directory where the skewT plots are to be placed.
  • -d dir: The directory where the ESC formatted sounding files to be plotted are located.
  • -w int: The frequency of the wind values to be kept in the plot. (i.e. -w 60 will keep all of the wind values that fall on the minute.)
  • -press high low: The range of pressure values to be on the plot. (i.e -press 1050 500 will show the pressure range of 1050 mb to 500 mb)
  • -temp low high: The range of temperature values to be on the plot. (i.e. -temp -5 35 will show the temperature range of -5°C to 35°C)
