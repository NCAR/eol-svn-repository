This dir contains test file input (*tsv), output CLASS files (*.cls),
and error logs (*errlog). These files along with the compareErrLogs script
can be used to regression test the SagarKanyaSounding_converter.pl converter. 
See the header of the S*.pl program for more details on what this tool
does and how it runs real-time via a crontab.
