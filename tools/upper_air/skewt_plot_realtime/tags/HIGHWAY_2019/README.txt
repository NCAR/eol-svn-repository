The real-time-plots directory contains the directory structure needed for running the real-time skew-t plotting software.
The real-time-plots/software directory contains the software for creating the skew-t plots from the ESC formatted files.
See the Real-time-Skew-T-Plotting.pdf file under docs for information on where in svn the software for the GTS_BUFR 
conversion can be found.
