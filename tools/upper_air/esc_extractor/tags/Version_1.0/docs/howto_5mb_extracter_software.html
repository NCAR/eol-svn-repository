Purpose

The purpose of this document is to describe the process and files for creating the 5 mb extraction software for soundings. The software reads CLASS formated data and generates new CLASS files with the surface data point and records for each pressure value that is a multiple of 5. If there is not a pressure value at the 5 mb interval, it is interpolated from the surrounding points.

Repository

The 5 mb extraction software is under revision control through subversion. (For more information on subversion see: How To: Subversion.) The repository can be found at:

/work/software/repository/subversion/tool/upper_air/composite/5mb/extracter

Directory Structure

bin The directory where the binaries are to be stored once they have been generated.
docs The directory where all of the documentation files for the software should be stored.
src The directory where all of the source and Makefiles should be stored for generating the binaries.

Software

There are several files in the src directory, but only three create the extract5mb binary (s2m.f, calcsm.f, gni.f). The extract5mb binary is also called s2m. I believe the others will make other 5 mb extracters for different types of soundings (dropsondes), but these haven't been checked yet.

To create the binary use the Makefile or use the command:

f77 -o ../bin/extract5mb s2m.f calcsm.f gni.f
