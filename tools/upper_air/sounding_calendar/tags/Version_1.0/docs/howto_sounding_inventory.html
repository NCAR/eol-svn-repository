Purpose

The purpose of this document is to describe the necessary steps for generating sounding inventories in an HTML calendar format.

Repository

This tool is under revision control using subversion. It can be found in:

http://svn.atd.ucar.edu/svn/dmg/tools/upper_air/sounding_calendar

Configuration File

This section defines the configuration file used by the script for generating the inventory calendars. It is a very specific file and needs to have all of the following information for the script to execute correctly.

The configuration file is to be named inv-network where network is the network name. The script uses the network as part of its output.

The file must define the following values:

  • source: The directory where the files to be inventoried are located.
  • dest: The directory where the inventory calendars are to be put once they have been created.
  • title: The title to be used on all of the calendars.
  • desc: The description of the calendar files.
  • stations: A space separated list of stations to be inventoried.
  • filename: The pattern used to find the correct files to be inventoried. Use the following patterns:
    • S+: for the station id
    • YYYY: for the four digit year
    • MM: for the two digit month
    • DD: for the two digit day
    • hh: for the two digit hour

Executing the Script

To run the script, run:

/net/work/bin/scripts/inventory/sounding_calendar_inventory.pl inv-network
