Use Case: Delete a Dataset from a Project

  1. The user wants to delete a dataset Delta from a project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The tool displays the editor for Zeta.
  4. The user finds Delta in the dataset list and selects the Delete option.
  5. The tool prompts the user to confirm with warnings.
  6. The user accepts the confirmation of deleting Delta.
  7. The tool removes the association between Delta and Zeta.
  8. The tool removes Delta from the database if it is no longer associated with a project.
  9. The tool regenerates the static HTML pages for Zeta.
  10. The tool refreshes the editor for Zeta.
