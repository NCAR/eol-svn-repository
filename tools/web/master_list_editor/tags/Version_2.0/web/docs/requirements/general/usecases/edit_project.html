Use Case: Edit a Project

  1. The user wants to edit project Zeta.
  2. The user goes to the Master List home page and selects Edit a Project.
  3. The tool displays the list of projects that can be edited.
  4. The user chooses project Zeta.
  5. The tool displays the project form with the values filled in for Zeta.
  6. The user changes the values as necessary for Zeta and clicks on the Next button.
  7. The tool displays the category selection page.
  8. The user updates the selection of categories for Zeta and selects the Finished button.
  9. The tool updates project Zeta in the database.
  10. The tool removes categories that are not longer selected for Zeta from the database.
  11. The tool adds associations for categories that were not originally associated with Zeta.
  12. The tool adds the associations for the categories that were not selected on the category selection page, but are the parent of a selected category.
  13. The tool removes datasets that are no longer associated Zeta because they no longer have a valid category for Zeta.
  14. The tool generates the static HTML pages for Zeta.
  15. The tool transfers control to the editor for Zeta.
