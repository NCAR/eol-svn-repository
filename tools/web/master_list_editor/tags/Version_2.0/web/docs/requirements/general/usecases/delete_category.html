Use Case: Delete a Category

  1. The user wants to delete the category Kappa from the database.
  2. The user goes to the Master List home page and selects Delete a Category option.
  3. The tool displays the list of categories that can be deleted (only categories that are not associated with a project [could still be associated to a dataset]).
  4. The user chooses Kappa and selects the Next option.
  5. The tool displays a confirmation dialog with warnings.
  6. The user confirms the deletion of Kappa.
  7. The tool removes any dataset associations with Kappa.
  8. The tool removes Kappa from the database.
  9. The tool displays a success dialog for the deletion of Kappa.
