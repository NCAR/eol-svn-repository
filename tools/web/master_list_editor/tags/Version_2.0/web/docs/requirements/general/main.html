The General Master List Editor

The General Master List (ML) is a tool for organizing data sets for a project in a way that is not available in the current data ordering system and is more useful to the end users. It groups the data sets by their categories for quicker access within the project.

The General ML was developed by Dan Sullivan (version 1.0 and 1.1). It is currently being controled by Joel Clawson and is up to version 1.2.

Relation to the DTS

The General ML is a small piece of the Data Tracking System. It handles assigning data sets to project's ML and providing other information that is useful to the user that is not typically part of a data set (if it is new, if the data has been updated, date the data set was made available, etc.).
