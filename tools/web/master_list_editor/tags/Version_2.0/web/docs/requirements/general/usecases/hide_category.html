Use Case: Hide a Category in a Project

  1. The user wants to hide category Kappa in project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The tool displays the editor for Zeta.
  4. The user finds the category Kappa and selects the Hide option.
  5. The tool marks Kappa as hidden for Zeta.
  6. The tool regenerates the static HTML pages for Zeta.
  7. The tool refreshes the editor for Zeta.
