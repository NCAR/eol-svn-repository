Use Case: Edit a Category

  1. The user wants to edit category Kappa.
  2. The user goes to the Master List home page and selects the Edit a Category option.
  3. The tool displays a list of categories in the database that are available to be edited.
  4. The user selects Kappa and clicks on the Next option.
  5. The tool displays the category form with the current values of Kappa filled in.
  6. The user updates the values in the form with the new values for Kappa and selects the Finished option.
  7. The tool updates Kappa in the database.
  8. The tool generates the static HTML pages for projects that are associated with Kappa.
  9. The tool displays the list of projects that had their Master List regenerated.
