Use Case: Adding an Existing Dataset to a Project

  1. The user wants to add an existing dataset Delta to project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The tool displays the editor for Zeta.
  4. The user selects the option for adding an existing dataset.
  5. The tool prompts for the dataset id of the existing dataset.
  6. The user enters the dataset id for Delta and selects the Accept option.
  7. The tool displays the information for Delta (which cannot be edited).
  8. The user selects the Associate with Project option.
  9. The tool creates the association between Delta and Zeta.
  10. The tool generates the static HTML pages for Zeta.
  11. The tool refreshes the editor for Zeta.
