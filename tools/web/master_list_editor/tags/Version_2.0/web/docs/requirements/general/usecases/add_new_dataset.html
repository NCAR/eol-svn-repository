Use Case: Add a New Dataset for a Project

  1. The user wants to add a new dataset Delta to a project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The tool displays the editor for Zeta.
  4. The use selects the Add New Dataset option.
  5. The tool displays the dataset form with the fields empty.
  6. The user fills in the form with the values of Delta and selects the Insert option.
  7. The tool inserts Delta into the database.
  8. The tool adds the association between Delta and its categories.
  9. The tool adds the association between Delta and Zeta.
  10. The tool regenerates the static HTML pages for Zeta.
  11. The tool refreshes the editor for Zeta.
