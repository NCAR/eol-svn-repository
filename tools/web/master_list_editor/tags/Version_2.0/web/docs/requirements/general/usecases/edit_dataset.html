Use Case: Edit a Dataset

  1. The user wants to edit a dataset Delta for a project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The tool displays the editor for Zeta.
  4. The user finds Delta in the dataset list and selects the Edit option.
  5. The tool displays the dataset form with the current values for Delta filled in.
  6. The user updates the form with the new values for Delta and selects the Update option.
  7. The tool updates Delta in the database.
  8. The tool updates the category associations for Delta that have been defined in the dataset form.
  9. The tool regenerates the static HTML pages for Zeta and any other project associated with Delta if non-project specific information was changed.
  10. The tool refreshes the editor for Zeta.
