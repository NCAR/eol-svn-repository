Use Case: Create a New Project

  1. The user wants to create a new project Zeta.
  2. The user goes to the Master List home page and selects Create a new Project.
  3. The tool displays the project form with empty fields.
  4. The user fills in the fields of the project form with the values for Zeta and clicks on the Next button.
  5. The tool displays the category selection page.
  6. The user selects/deselects the categories to be associated with Zeta as necessary.
  7. The user selections the Finished button.
  8. The tool creates the Zeta project in the database.
  9. The tool adds the associations for each category that was selected on the category selection page to Zeta.
  10. The tool adds the associations for the categories that were not selected on the category selection page, but are the parent of a selected category.
  11. The tool generates the static HTML pages for Zeta.
  12. The tool transfers control to the editor for Zeta.
