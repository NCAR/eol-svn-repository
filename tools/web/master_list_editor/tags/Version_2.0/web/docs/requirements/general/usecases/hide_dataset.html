Use Case: Hide a Dataset for a Project

  1. The user wants to hide dataset Delta in project Zeta.
  2. The user goes to the Master List home page and selects the editor for Zeta.
  3. The user finds dataset Delta in the dataset list and selects the Hide option.
  4. The tool sets the flag to hide Delta in Zeta.
  5. The tool regenerates the static HTML pages for Zeta.
  6. The tool refreshes the editor for Zeta.
