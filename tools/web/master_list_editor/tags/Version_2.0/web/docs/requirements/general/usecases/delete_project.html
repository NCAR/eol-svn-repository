Delete a Project

  1. The user wants to delete the entire Master List for project Zeta.
  2. The user goes to the Master List home page and selects Delete a Project.
  3. The tool displays the list of projects in the database and the list of projects that are in progress of being cloned.
  4. The user chooses project Zeta to be deleted.
  5. The tool displays a confirmation dialog with warnings about the deletion.
  6. The user confirms the deletion.
  7. The tool removes all category associations with Zeta.
  8. The tool removes all dataset associations with Zeta.
  9. The tool removes all datasets that are no longer associated with a project. (These were the datasets that were only associated with Zeta.)
  10. The tool removes the static HTML pages from the system.
  11. The tool removes the project Zeta from the database.
  12. The tool displays a success dialog to the user.
