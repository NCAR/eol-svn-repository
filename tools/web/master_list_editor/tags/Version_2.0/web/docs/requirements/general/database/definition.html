Database Model plugin_tmp.xml

category
The category table contains all of the categories that can be used in a master list. It uses a parent id to define the heirarchy.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
category_id INTEGER PK NN UNSIGNED   The id number for the category AI
name VARCHAR(63)   NN     The unique name for the category  
parent_id INTEGER     UNSIGNED NULL The id of the category that is the parent of this category.  
IndexName IndexType Columns
PRIMARY PRIMARY category_id
uniqueName Unique Index name
parentFK Index parent_id


dataset
The dataset table contains the dataset information for the master list.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
dataset_id VARCHAR(15) PK NN     The identifier for the dataset (should be the same as zedi) AI
name VARCHAR(255)   NN     The name of the dataset (should be the same as zedi)  
url VARCHAR(255)   NN     The url that accesses the dataset  
doc_url VARCHAR(255)   NN     The url that accesses the documentation for the dataset  
expected_date DATE         The date when the data is expected to be ingested  
entered_date DATE         The date when the dataset is put online  
update_flag TINYINT(1)   NN UNSIGNED 0 A flag that signifies if the dataset has been updated  
IndexName IndexType Columns
PRIMARY PRIMARY dataset_id


dataset_category
The dataset_category associates categories to datasets. This will make a single dataset appear in all associates categories for a project.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
dataset_id VARCHAR(15) PK NN     The dataset being assigned a category  
category_id INTEGER PK NN UNSIGNED   The category being assigned to a dataset  
IndexName IndexType Columns
PRIMARY PRIMARY dataset_id
category_id
datasetIdFK Index dataset_id
categoryIdFK Index category_id


dataset_project
The dataset_project provides the association between projects and datasets. It provides the means to flag the dataset for the project.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
dataset_id VARCHAR(15) PK NN     The dataset being assigned to a project  
project_id VARCHAR(15) PK NN     The project being assigned a dataset  
in_progress_flag TINYINT(1)   NN UNSIGNED 0 A flag to signify that the dataset is in progress for the project  
hide_flag TINYINT(1)   NN UNSIGNED 0 A flag to hide all instances of the dataset for the project  
IndexName IndexType Columns
PRIMARY PRIMARY dataset_id
project_id
datasetIdFK Index dataset_id
projectIdFK Index project_id


project
The project table contains the projects that have a general master list.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
project_id VARCHAR(15) PK NN     Uniquely defines the project (should be the same as zedi)  
display_name VARCHAR(63)   NN     The name of the project to display to the user  
system_directory VARCHAR(255)   NN     The directory on the local system where the static HTML is to be written  
url VARCHAR(255)   NN     The url where the master list will be accessed  
new_length SMALLINT(2)   NN UNSIGNED 90 The number of days that a dataset is considered "new"  
IndexName IndexType Columns
PRIMARY PRIMARY project_id
uniqueSystemDirectory Unique Index system_directory
uniqueURL Unique Index url


project_category
The project_category assigns categories to a project. The assigned categories make up the list on the left side of the general master list.
ColumnName DataType PrimaryKey   NotNull   Flags Default Value Comment AutoInc
project_id VARCHAR(15) PK NN     The project being assigned a category  
category_id INTEGER PK NN UNSIGNED   The category being associated to a project  
hide_flag TINYINT(1)   NN UNSIGNED   A flag that hides the category in the master list without removing the associations  
IndexName IndexType Columns
PRIMARY PRIMARY project_id
category_id
projectIdFK Index project_id
categoryIdFK Index category_id
