Use Case: Create a New Category

  1. The user wants to create a new category Kappa.
  2. The user goes to the Master List home page and selects the option Create a Category.
  3. The tool displays the category form with empty fields.
  4. The user enters the values for Kappa in the form and clicks on the Finished option.
  5. The tool inserts Kappa into the database.
  6. The tool transfers control back to the Master List home page.
