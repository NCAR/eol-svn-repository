Provides a set of states used by the General Master List.

All of the states defined in this package are Concrete States for the State pattern defined in the Design Patterns by Gamma, et. al.

There are two sets of states. They are:

  • Category States: that manipulate Categories in the database.
  • Dataset States: that manipulate Datasets in the database.
  • Project States: that manipulate Projects in the database.
  • Delete States: that delete entries from the database.
  • Edit States: that edits entries from the database.
  • List States: that list entries in the database.
