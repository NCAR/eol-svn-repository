IVEN Requirements

Required Information

Project

  • Project ID
  • Title
  • Active Flag
  • Dataset Prefix
  • Lat/Lon Box
  • Start & End dates
  • Charge Number
  • Set of Notes

Products

A product can be either a Composite dataset or a Group.

  • Product Id
  • Title
  • Status
  • Set of Datasets
    • Excluded Flag
  • Set of Notes

Dataset

  • Dataset Id
  • Title
  • Process Contact
  • Status
  • Work Directory
  • Final Data Directory
  • Plot Link
  • Repository Directory
  • Station List
  • How To Link
  • Set of Docs for the Dataset
  • Set of Source Datasets
    • Source Dataset Id
    • Source Title
    • Set of Source Docs
    • UTC Offset of Source Data
    • DST Flag for Source Data
    • Collection Time of Source Data
    • Source of Time Info for Source Data
    • Set of Source Notes
  • Set of Processing Notes
  • Set of Associated Products

Notes

  • Note Id
  • Author
  • Timestamp
  • Text

Other Needs

  • List of Users that can be assigned a task (must be able to handle users that are no longer active)
  • List of Statuses that can be assigned to a task

Required Views

  • List of Active Projects
    • Displays all active projects with the project id, and full name of the project.
  • Project Summary
    • Displays general project information including: start/end dates, lat/lon box, charge number, dataset prefix, etc.
    • Displays a table of the products displaying the name, status, progress bar of product datasets, excluded dataset count, and total dataset count.
    • Displays an expanded list of products with the title, status, and all associated datasets with their name, process contact, and status.
  • Group Summary
    • Displays general project information for the group including: start/end dates, lat/lon box, charge number, dataset prefix, etc.
    • Displays the group title, status, and the list of notes.
    • Displays the list of datasets in the group including the title and dataset notes.
  • Dataset Summary (Includes Composite)
    • Displays general project information for the dataset including: start/end dates, lat/lon box, charge number, dataset prefix, etc.
    • Displays the dataset title, status, processing contact, work directory, final data directory, plot link, repository directory, station list, how to link, list of dataset readmes, list of source information (title, readmes, UTC offset, DST Flag, collection time, and source of time information), and the set of notes.
    • Displays the list of source dataset used for creating the dataset.
    • Displays the list of associated products.

Required Functionality

  • General Functionality
    • Dataset ids must be unique as defined by the data archive system.
    • All changes to a dataset must be applied to all projects and products associated with the dataset.
  • Updating Project Notes
  • Creating a new Group
    • Defines the following fields for the Group
      • Group Name (Must be unique for the project set by tool using the currently selected project)
      • Status
      • Set of Notes for the group
    • Inserts values into the DTS database:
      • Database auto generates a new group id for the entry.
      • Assigns the group name,status, and project to the entry.
      • Creates new note(s) and associates it(them) to the dataset.
    • Edit a Group
      • Defines the same fields as in Create a new Group
      • Cannot change the group id or project.
      • Updates the values in the DTS database.
      • Create/Update the notes as needed.
    • Create a new Composite
      • Defines the following fields for the Dataset
        • Dataset Id - Must not yet exist in the data archive system
        • Title
        • Status
        • Set of Notes
      • Inserts values into the data archive system database:
        • Defaults the lat/lon box to the lat/lon box for the project
        • Defaults the start/end dates to the start/end dates for the project
        • Defaults the hidden flag to true
        • Assigns the dataset id and name from the form
        • Associated the dataset to the project
      • Inserts values into the DTS database:
        • Assigns the status to the dataset
        • Creates and assigns the notes to the dataset.
    • Associate a Dataset/Group and a Source Dataset
      • Requires the Source Dataset to already exist
      • Defines the following fields for the Source Dataset (will need to use existing information for source dataset if it already exists)
        • UTC offset
        • Daylight Saving Switch Flag
        • Note for Collection Time of Data
        • Note for the Source of the Time Information
      • Inserts the source dataset information and associates the source dataset with the dataset/group in the DTS database.
    • Remove a Dataset/Group and Source Dataset Association
      • Removes the association between the source dataset and the dataset/group. (It does not remove any other information)
    • Exclude/Re-include a Source Dataset
      • Mark the exclude flag for the source dataset and dataset/group in the DTS database.
    • Edit a Dataset (includes Composites)
      • Defines the following fields for the Dataset:
        • Work Directory
        • Final Data Directory
        • Plot Link
        • Repository Directory
        • Station List
        • How To Link
        • Status
        • Processing Contact
        • Questions Flag
        • Processing Notes
    • Updates all of the values in the fields into the DTS database.
    • Create/Update notes as necessary.

Purposely Omitted Functionality

  • Adding/Updating/Deleting a Project or any information (done in other tools)
  • Creating/Deleing a new non-Composite dataset (done in other tools)
  • Editing dataset information that is not related to processing (done in other tools)
  • Deleting a Composite/Group that has associated source dataset
