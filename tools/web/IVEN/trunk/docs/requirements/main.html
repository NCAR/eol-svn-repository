DTS Version 2.0

The Data Tracking System Version 2.0 is a set of new databases that combine and expand on the Version 1.0 databases. The new databases utililze the information in the data archive system to prevent to duplication of information as well. It upgrades and expandes the current tools to handle the changes.

The Master Lists are different than the other tools. They require more work than the other tools before they are ready to be merged into the Version 2.0 system. For this reason, the Master Lists will go be merged in three phases.

  1. Upgrading the existing stand alone databases and associated tools to work in a cleaner fashion (both general and arctic)
  2. Merge the general and arctic databases into a single stand alone master list database and merge the editors into a single tool
  3. Merge the stand alone master list database into the new master list database that fits within the new DTS/data archive system structure
