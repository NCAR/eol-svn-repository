DLN Objective

The Data Loading Notes is to provide a useful tool for the Data Management Group to maintain and manage the loading and checking of data sets. It is to contain the location of the data to be loaded, the final archive location for the data archive system, the person assigned to loading the data set, and the person who is assigned to check the data set.

Problem Statements / Upgrades

Change labels (as necessary) to be more intuitive to users and prevent confusion. Remove functionality that is either not used or does not make sense in this context.
