The Data Loading Notes (DLN)

The Data Loading Notes (DLN) is a tool for maintaining the information needed to load and check a data set. It maintains a set of notes, locations for the data, and which person was assigned to load and check the dataset.

The DLN was developed by Dan Sullivan (version 1.0). It is currently under control by Joel Clawson and has up to version 1.1.

Relation to the DTS

The DLN is the piece of the Data Tracking System that handles the loading and approving tasks along with the ingest task to a lesser extent. Versions 1.x are in the DTS architecture Version 1. The planned version 2.x are in the DTS architecture Version 2. The DLN fits into the DTS architecture Version 3 as part of the DTS web interface. At this point the DLN stand-alone tool will be retired as it is no longer needed.
