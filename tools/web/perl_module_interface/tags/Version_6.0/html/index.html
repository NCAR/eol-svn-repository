Perl Conversion Module Interface

There are a number of utilities functions defined as part of the Perl conversion module library. These are independent of the different format conversions. This lists many of these functions and provides a web based interface to them.

Web Scripts
  WebInterface The super class for the web scripts. It defines the output format and all the common links and documentation for the scripts.
 
Air Density (doc) Calculate the air density from the pressure and temperature.
Altitude (doc) Calculate the altitude from the previous pressure, temperature, dew point, and altitude and the current pressure, temperautre and dew point.
Dew Point (doc) Calculate the dew point from the temperature and relative humidity.
Mixing Ratio (doc) Calculate the mixing ratio from the pressure and vapor pressure.
Relative Humidity (doc) Calculate the relative humidity from the temperature and dew point.
Sea Level Pressure (doc) Calculate the sea level pressure from the pressure, elevation, temperature, and dew point.
Specific Humidity (doc) Calculate the specific humidity from the pressure and dew point.
Station Pressure (doc) Calculate the station pressure from the altimeter and elevation.
UV Winds (doc) Calculate the u and v wind components from the wind speed and direction.
Vapor Pressure (doc) Calculate the [saturated] vapor pressure from the [temperature] dew point.
Virtual Temperature (doc) Calculate the virtual temperature from the temperature and the mixing ratio.
Wind Speed/Direction (doc) Calculate the wind speed and direction from the u and v wind components.
 
Angles (doc) Convert an angle value to a different unit
Area (doc) Convert an area value to a different unit.
Flow (doc) Convert a flow value to a different unit.
Length (doc) Convert a length value to a different unit.
Pressure (doc) Convert a pressure value to a different unit.
Radiation (doc) Convert a radiation value to a different unit.
Temperature (doc) Convert a temperature value to a different unit.
Time (doc) Convert a time value to a different unit.
Velocity (doc) Convert a velocity value to a different unit.
Volume (doc) Convert a velocity value to a different unit.
 
[Julian] Date (doc) Convert a date value to a different format.
Latitude/Longitude (doc) Convert a latitude/longitude value to a different format.
State Code (doc) Convert a state code to its abbreviation or vice versa.
