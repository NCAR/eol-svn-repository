The DPG Project Page

With the new database setup, it will be possible to create a script that will automatically generate our DPG Project pages. This could define the basic layout of the page, which would now be consistantly designed between the projects and would remind (force) us to enter certain things for every project. Much of the currently displayed information would be read from jedi like the lat/lon box, the begin and end dates, and the project description.

This would now require a script to edit and would be similar to editing the notes for IVEN. This script and the part for IVEN that reads the project notes would now reference the same information in the database, thus reducing duplication of work.
