Data Tracking System Requirements

This documentation contains the detailed requirements for the Data Tracking System.

General Functionality

  • Create A New Dataset
  • Clone A Dataset
  • Edit An Existing Dataset

Restricted Functionality

  • Delete a Dataset
  • Create a New Project
  • Edit an Existing Project
  • Create a New User
  • Edit an Existing User

Omitted Functionality

  • Delete a Project
  • Delete a User

Views

  • List of Projects
  • List of Datasets by Project
  • List of Processed Datasets by Project
  • List of Composite Dataset by Project
  • List of Datasets by User
  • List of Users
  • Summary of a Project
  • Summary of a Dataset

Editors

  • Dataset Ingest Task Editor
  • Dataset Process Task Editor
  • Dataset Load/Approve Task Editor
  • Project Editor
  • User Editor

Dataset Definition

  • Dataset Identifier
  • Dataset Title
  • Dataset Type (Source, Processed, Composite, Link, Imagery)
  • Set of Associated Projects
  • Set of General Notes
  • User Who Entered the Dataset
  • Date the Dataset was Entered
  • Set of Remote URLs
  • Internal Dataset Contact
  • External Dataset (Source) Contact
  • Documentation Flag
  • Master List Flag
  • Ingest Task Information
    • Ingest User
    • Ingest Status
    • Ingest Location
    • Ingest Type (FTP, Email, Manual Download, CD, etc.)
    • Ingest Specific Notes
  • Process Task Information
    • Working Directory
    • Repository
    • Repository Revision Number
    • Final Data Location
    • Station List Location
    • How To URL
    • Data Plots URL
    • Question/Issue Flag
    • Platform
    • Set of Source Datasets
      • Set of Documentaiton Files
      • UTC Offset Notes
      • DST Notes
      • Notes on Collection Times
      • Notes for Data Source Information
    • Processer
    • Process Status
    • Processing Priority
    • Processing Specific Notes
  • Load Task Information
    • Loader
    • Load Status
    • Location of the Data to be Loaded
    • Final Archive Location
    • Load Specific Notes
  • Approve Task Information
    • Approver
    • Approve Status
    • Approve Specific Notes

Project Definition

  • Project Identifier
  • Active Flag
  • Set of Project Notes

User Definition

  • Identification Number
  • User Name
  • Password
  • Email Address
  • Active Flag
  • Set of User Roles
