You have selected a dataset outside the EOL archive. to download via FTP. If you are behind a firewall or using a DSL connection and using Internet Explorer you must be in "Passive FTP" mode. If you are already in passive mode click the dataset name below to continue. If not or unsure follow the instructions below.




How to use passive mode in IE:
  • Select Tools from the menu.
  • Select Internet Options...
  • Click on the Advanced tab.
  • Be sure the Use Passive FTP... option is selected.
  • Click OK and proceed to the FTP site by clicking on the link above.
