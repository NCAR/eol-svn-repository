formit* - script to generated web page and move to proper BAMEX location. Runs *form_webPage_output.pl PERL code.
web.log  - generated during run. See formit script.

AllColns_form_webPage_output.pl* - s/w that will output much more stn info and
             produce a much larger web page. This is a better script if the input
             station list is not too large. If you have thousands of stations, you
             probably need to use form_webPage_output.pl s/w to control output web page size.

form_webPage_output.pl* - code that generates final web page and calls C code to calc distances.
                          Generally run this script, particularly if have thousands of stns in
                          stationCD.out file.

det_closest_stn* - C code that calc's distances and is called by form_webPage_output.pl.  
                   Uses same dist calc as HQC.

BAMEX_stn_list.ID - stationCD.out file sorted by ID. This is the input to the *form_webPage_output.pl PERL code.

Closest_stns_LIST.out - generated by form_webPage_output.pl.
Closest_stns_MATRIX.out - generated by form_webPage_output.pl and is a matrix containing the distance
                          between every station in table form.
