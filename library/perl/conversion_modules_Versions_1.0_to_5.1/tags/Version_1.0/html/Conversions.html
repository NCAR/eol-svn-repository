Conversions Module

Module: Conversions.pm

Location: /home/jclawson/conversion_modules

Summary: The Conversions module is a collection of static functions used for calculating and converting values. It also contains functions for accessing static values such as the QCF missing value and the different QCF flags.

Use: In a perl script, import the Conversions modules with the following commands:
use lib "location";
use Conversions;

Convert Temperature

float convertTemperature(float temp, String inUnits, String outUnits)

Convert a temperature value from one unit to another unit. Acceptable values for the units are: C, F, K

Parameters:
temp: The temperature value to be converted.
inUnits: The unit of the temperature to be converted.
outUnits: The unit of temperature to convert the temp parameter.

Returns: The temperature in outUnits.
